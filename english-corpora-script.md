# Using English Corpora to get KWICs

From [English Corpora main page](https://www.english-corpora.org/)

- iWeb is near the tob of the table of corpora
- click it
- if not already logged in, do so (person profile icon in menu)
- click search tab
- click word tab above search box
- enter a word in box e.g. "snow"
- click "see detailed info"
- on right hand side, in top line of links, click KWIC
- this gives a KWIC, which you can sort 


# Using English Corpora to get collocate counts

From [English Corpora main page](https://www.english-corpora.org/)

- iWeb is near the tob of the table of corpora
- click it
- if not already logged in, do so (person profile icon in menu)
- click search tab
- click word tab above search box
- enter a word in box e.g. "snow"
- click "see detailed info"
- on right hand side, click collocates
- this gives a list of collocates e.g. nouns
